# 02. Climbing stairs  `[easy]`

Each step you can climb 1 or 2 stairs. Return the number of distinct ways to reach the top of `n` stairs.

def climb_stairs(n: int) -> int: ...

## Examples

| n | ways |

| 2 | 2 |
